You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /github -- parses and returns information about the specified query. Only works for users/USERNAME/repos.
  • /babynames -- returns a random baby name based on parameters: gender -> boy or girl, popular -> true or false
    • Example: /babynames?gender=girl&popular=true
  • /convertcurrency -- takes an amount, source currency, and a target currency. Returns the conversion.
    • Example: /convertcurrency?amount=100&source=USD&target=EUR
